Sistema de Inventario Médico

Proyecto para la gestión de productos, stock y control de inventario

Inicio Objetivo Módulos Contacto

Inicio

Este repositorio contiene el desarrollo de un sistema de inventario médico, diseñado para organizar productos, controlar cantidades disponibles y facilitar el registro de entradas y salidas dentro de un entorno clínico o administrativo.

Objetivo del Proyecto

Desarrollar una solución que permita administrar de forma ordenada el inventario médico, evitando pérdidas de información, mejorando el control de stock y facilitando la consulta de productos disponibles.

Módulos Principales

Registro de productos

Permite ingresar nuevos productos médicos con su nombre, cantidad y categoría.

Control de stock

Permite visualizar las existencias disponibles y actualizar cantidades.

Despacho

Registra la salida de productos del inventario.

Reportes

Facilita la revisión general del inventario y los movimientos realizados.
